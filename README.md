# FYP 2026 — Scholarship & Internship Portal

A web application to guide bachelor graduates in finding scholarships and internships.

## Tech Stack
- Frontend: React.js
- Backend: Python (FastAPI/Flask)
- Database: ChromaDB

## Features
- Scholarship listings
- Internship portal
- AI-based recommendations

## Setup
### Backend
cd backend
pip install -r requirements.txt
python app.py

### Frontend
cd frontend
npm install
npm run dev
